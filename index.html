Get Started with Domino

Visual Workflows for Everyone

Domino brings an intuitive Graphical User Interface that facilitates creating, editing and monitoring any type of Workflow, from data processing to machine learning.

Reusability, Reproducibility and Collaboration

Domino proposes a standard way of writing and publishing functional Pieces, which follows good practices for distribution, documentation and data modeling.

Powered by Apache Airflow

Domino is powered by Apache Airflow for top-tier workflows scheduling and monitoring.
